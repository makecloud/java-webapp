为线下屏幕赋能科技

OOHLink自主研发国内首创的户外广告程序化交易平台，整合线下丰富的媒体资源，通过平台标准化产品和大数据处理技术，平台独特的个性化投放模式和智能高效的精准播控系统，为传统户外广告快速赋能，使户外广告媒体达到媒体资源程序化购买，让线下广告营销更加智能、便捷。

数据驱动的户外3.0时代

OOHlink依托数据搜集入口，对接多个数据源，通过营销将品牌推广，将消费用户引流，帮助商户获客，基于线下投放产生的数据分析，进行线上精准营销，为广告主提供智能化、精准的户外广告投放，并通过线上线下跨屏打通以增强广告的曝光效果，真正实现户外广告的品效合一。

场景沉浸式的闭环营销模式

OOHlink独特的营销体验，链接现实与虚拟，打通线上与线下，同时科技的被动感、场景的沉浸式营销搭配了直播、互动化、内容化等多元特征，共同构筑了户外媒体的独特价值，线下+线上联动营销让营销信息更有效的传达。

蜜色之吻

伴随着时代发展，线上线下相结合的营销方式出现，主流消费群体更倾向于线上消费（即电商）。熟悉网购的多是年轻消费者，根据数据显示，这些新生代的年轻消费群体很少去到内衣店消费，更多是采取网购的形式。 蜜色之吻如何在众内衣、众电商中脱颖而出，是我们此次传播的重点，通过本次广告传播挖掘关注内衣、有购买欲望的目标人群，进行精准投放。
